アクセス申請 最初にやること

Brain はプライベートリポジトリのため、インストール前に管理者から招待を受ける必要があります。
GitHub ユーザー名を入力して申請ボタンを押してください。

github.com/

GitHubアカウントをお持ちでない方は先に github.com で無料登録してください。申請後 GitHub からメールが届きます。「Accept invitation」を押してから Step 1 へ。

💻

Mac をお使いの方向けの手順です。
申請が完了したら、以下のステップを順番に進めてください。

1
✉️ 招待メールを承認する

管理者があなたの GitHub アカウントを招待します。
GitHub からメールが届いたら、承認してください。

招待の承認手順：

1
GitHub から 「You have been invited to collaborate」 という件名のメールが届きます
2
メール内の 「Accept invitation」 ボタンをクリックしてください
3
ブラウザが開き 「You now have access to RYKNSH/ryknsh-brain」 と表示されれば完了です
⚠️ ボタンが押せない・メールが届かない場合：
https://github.com/notifications?query=reason%3Ainvitation
※ GitHub にログインした状態で開いてください。
2
🤖 Claude アプリをインストール

Anthropic 社の AI「Claude」の公式アプリをダウンロードします。

✦
Claude for Mac claude.ai/download
ダウンロード →
💡 インストール後：アプリを起動してアカウントにサインインしておいてください。
3
🖥️ ターミナルを開く

「ターミナル」はMacに最初から入っているアプリです。
コマンドを入力して実行するための黒い画面です。

ターミナルの開き方
  1. キーボードの ⌘（command）+ スペース を同時に押す
  2. 「ターミナル」と入力する
  3. return（Enter） を押す
💡 ヒント：黒または白の画面が開けば成功です。
4
🍺 Homebrew をインストール

Homebrew は「アプリを簡単にインストールできる便利ツール」です。
下のコマンドをコピーしてターミナルに貼り付け、return を押してください。

コマンド
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
何が起きる？ パスワードを聞かれたら Mac のログインパスワードを入力してください（入力中は画面に何も表示されませんが正常です）。数分かかります。
5
🐙 GitHub CLI をインストール

ターミナルから GitHub を操作するためのツールです。プライベートリポジトリへのアクセスに必要です。

コマンド
brew install gh
何が起きる？ GitHub CLI のダウンロードとインストールが自動で行われます。1〜2 分で完了します。
6
🔑 GitHub にログイン

ターミナルから GitHub アカウントにログインします。

コマンド
gh auth login

コマンド実行後、順番に選択・入力していきます：

1
? What account do you want to log into? → GitHub.com を選択して return
2
? What is your preferred protocol? → HTTPS を選択して return
3
? Authenticate Git with your GitHub credentials? → Yes を選択して return
4
? How would you like to authenticate? → Login with a web browser を選択して return
5
画面に 8桁のコード が表示されます。メモしてから return を押してください。ブラウザが開きます。
6
ブラウザでコードを入力して 「Authorize」 をクリックすればログイン完了です。
完了の確認：ターミナルに「Logged in as [ユーザー名]」と表示されれば成功です。
7
🧠 Brain をインストール

Brain のコア（AIスキル・コマンド・環境設定）をまとめて導入します。

コマンド（これだけ！）
curl -fsSL https://raw.githubusercontent.com/RYKNSH/setup/main/bootstrap.sh | bash
何が起きる？ ~/.claude/ に Brain がクローンされ、以下が自動で行われます：
  • Node.js・1Password CLI・Google Workspace CLI などの必須ツールをインストール
  • マシン ID 登録：このマシン固有の識別情報（machine.conf）を自動生成
  • 27スキル・26コマンド・フックを展開
  • 診断スクリプト（brain-doctor.sh）を配備
「Brain デプロイ完了」と表示されれば Step 7 は完了です。
⚡ 途中で Action needed が表示されたら：ターミナル最後の「Action needed:」行に従って、1Password Desktop → Settings → Developer → Command Line Interface を ON、および gws auth login などを実行してください。
8
🚀 初回セッションを起動する

Claude アプリを開いて最初のセッションを起動します。
/ry-in を実行すると、環境の自動チェックと残りのセットアップが完了します。

Claude のチャット欄に入力
/ry-in

初回チェックインで自動実行されること：

1
Brain 同期：最新のスキル・コマンドを自動取得
2
環境ドリフト検出：CLI（op / gh / node / gws）と MCP の状態を確認
3
MCP セットアップ案内：Notion など未接続の統合があれば承認手順を提示
🔗 Notion 連携の承認：Brain は Notion MCP が必須です。/ry-in が「未接続」と表示したら、Claude アプリ → Settings → Integrations → Notion → Connect から OAuth を承認してください。
完了の確認：「✅ checkin 完了」と「CLI 4/4 OK」が表示されれば、すべてのセットアップが完了しています。
✓

セットアップ完了！

Brain が Claude アプリ上で動作する準備が整いました。
毎回のセッションは /ry-in で開始し、/ry-out で終了します。

セッション
/ry-in — セッション開始 /ry-out — セッション終了 /ry-pm — プロジェクト管理 /ry-init — プロジェクト初期化
マーケティング
/ry-lp — LP 戦略・コピー /ry-craft — LP HTML 実装 /ry-ads — 広告設計・コピー /ry-webinar — ウェビナー台本
クリエイティブ
/ry-slide — スライド生成 /ry-art — ビジュアル生成 /ry-content — SNS コンテンツ /ry-page — エディトリアル
品質・分析
/ry-review — コードレビュー /ry-sim — ペルソナ検証 /ry-debate — 多角ディベート /ry-think — 第一原理分解
🩺 Brain の健康チェック いつでも実行可

接続・認証・テレメトリに問題が起きたと感じたら、診断スクリプトを実行してください。マシン ID・push 権限・フック状態をまとめて確認できます。

ターミナルで実行
bash ~/.claude/admin/brain-doctor.sh
🔄 Brain を最新版に更新する 新バージョンが出たら

管理者から「アップデートあり」と連絡を受けたら、以下のコマンドを実行してください。新スキルや改善が自動で反映されます。

ターミナルで実行
cd ~/.claude/brain && git pull origin main && bash install.sh
