RYKNSH Brain Setup

AIエージェントを
セットアップする

コードの知識は不要です。
手順通りにコピペするだけで完了します。

所要時間：約 10〜15 分
🔐
アクセス申請 最初にやること

Brain はプライベートリポジトリのため、インストール前に管理者から招待を受ける必要があります。
GitHub ユーザー名を入力して申請ボタンを押してください。招待メールが届いたら次のステップへ進んでください。

github.com/

GitHubアカウントをお持ちでない方は、先に github.com で無料登録してからこちらに戻ってください。
申請が完了したら GitHub からメールが届きます。「Accept invitation」を押して承認してからStep 1へ進んでください。

💻

Mac をお使いの方向けの手順です。
申請が完了したら、以下のステップを順番に進めてください。

1
✉️ 招待メールを承認する

管理者があなたの GitHub アカウントを招待します。
GitHub からメールが届いたら、承認してください。

招待の承認手順：

1
GitHub から 「You have been invited to collaborate」 という件名のメールが届きます
2
メール内の 「Accept invitation」 ボタンをクリックしてください
3
ブラウザが開き 「You now have access to RYKNSH/ryknsh-brain」 と表示されれば完了です
⚠️ ボタンが押せない・メールが届かない場合：下のリンクをブラウザで開いて承認できます。
https://github.com/RYKNSH/ryknsh-brain/invitations
※ GitHub にログインした状態で開いてください。
2
🤖 Claude アプリをインストール

Anthropic 社の AI「Claude」の公式アプリをダウンロードします。
下のボタンをクリックして、ページの指示に従ってインストールしてください。

✦
Claude for Mac claude.ai/download
ダウンロード →
💡 インストール後：アプリを起動してアカウントにサインインしておいてください。
3
🖥️ ターミナルを開く

「ターミナル」はMacに最初から入っているアプリです。
コマンドを入力して実行するための黒い画面です。以下の手順で開いてください。

ターミナルの開き方
  1. キーボードの ⌘（command）+ スペース を同時に押す
  2. 「ターミナル」と入力する
  3. return（Enter） を押す
💡 ヒント：黒または白の画面が開けば成功です。文字が入力できる状態になっています。
4
🍺 Homebrew をインストール

Homebrew は「アプリを簡単にインストールできる便利ツール」です。
下のコマンドをコピーしてターミナルに貼り付け、return を押してください。

コマンド
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
何が起きる？ パスワードを聞かれたら Mac のログインパスワードを入力してください（入力中は画面に何も表示されませんが正常です）。インストールが完了するまで数分かかります。
5
🐙 GitHub CLI をインストール

GitHub CLI はターミナルから GitHub を操作するためのツールです。
プライベートリポジトリへのアクセスに必要です。下のコマンドをコピーして実行してください。

コマンド
brew install gh
何が起きる？ GitHub CLI のダウンロードとインストールが自動で行われます。1〜2 分で完了します。
6
🔑 GitHub にログイン

ターミナルから GitHub アカウントにログインします。
下のコマンドを実行して、画面の指示に従ってください。

コマンド
gh auth login

コマンド実行後、順番に選択・入力していきます：

1
? What account do you want to log into? → GitHub.com を選択して return
2
? What is your preferred protocol? → HTTPS を選択して return
3
? Authenticate Git with your GitHub credentials? → Yes を選択して return
4
? How would you like to authenticate? → Login with a web browser を選択して return
5
画面に 8桁のコード が表示されます。メモしてから return を押してください。ブラウザが開きます。
6
ブラウザでコードを入力して 「Authorize」 をクリックすればログイン完了です。
完了の確認：ターミナルに「Logged in as [ユーザー名]」と表示されれば成功です。
7
🧠 Brain をインストール

いよいよ最後のステップです。
下のコマンドをコピーしてターミナルに貼り付け、return を押してください。

コマンド（これだけ！）
curl -fsSL https://raw.githubusercontent.com/RYKNSH/setup/main/bootstrap.sh | bash
何が起きる？ AIスキルや設定ファイルが自動でインストールされます。「Installation Complete」と表示されれば完了です。
✓

セットアップ完了！

Claude アプリを開けば、Brain が使えるようになっています。
セッションを始めるときは /ry-in と入力してください。

よく使うコマンド

/ry-in　　— セッション開始 /ry-out　 — セッション終了 /go　　　 — タスク実行
